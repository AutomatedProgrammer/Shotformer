# Shotformer
Basic platformer with a gun made on Python. Although I'd say its more of a proof of concept since there isn't really a game, just a handful of mechanics for a platformer. This project was done as a group project with Time Traveller, AsianPersuasion, and Monatela to learn the basics of Git mainly. Unfortunately due to school, Frige was the only one left to finish the project so this is the result.

Game Mechanics: The-Frige and AsianPersuasion

Level: TimeTraveller

Sprites: Monatela

Arrow keys to move. R to reload.
